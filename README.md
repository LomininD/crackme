# CrackMe

>*Данная версия readme промежуточная, более полное описание будет позже.*

Парный проект, заключающийся в создании простой программы, запрашивающей пароль, и последующего взлома. Выполнялся в операционной системе MS-DOS. В качестве методов взлома использовались классическое переполнение буфера, самомодифицирующийся код, перехват системных прерываний и статическая бинарная инструментация с помощью программы-патчера на C.
